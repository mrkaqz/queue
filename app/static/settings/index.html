Setup guide:
1. Go to developers.facebook.com → Create App → Business
2. Left sidebar → Add product → choose Messenger → Set up
3. Access Tokens → select your Facebook Page → copy Page Access Token → paste above → click 🔍 Test to verify
4. Fill in App Secret + Webhook Verify Token → Save Settings
5. Messenger → Webhooks → Add Callback URL → paste the Webhook URL above + your Verify Token → Verify and Save
6. Subscribe the webhook to the messages event
7. Switch app to Live Mode for all customers (Dev Mode = admins/testers only)
⚠️ Public HTTPS URL required — the Webhook URL must be reachable from the internet (not a private LAN IP)
